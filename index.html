    20.5.2 Properties of the Error Constructor

    20.5.2.1 Error.captureStackTrace ( O [ , limit ] )

    1. If O is not an Object, throw a TypeError exception.
    2. If IsCallable(limit) is true, then
      1. Let string be an implementation-defined string that represents the current stack trace, with all stack frames above the topmost call to limit, including that call, omitted from the stack trace.
      2. NOTE: This can be used to hide implementation details on the stack trace that aren't useful to user.
    3. Else,
      1. Let string be an implementation-defined string that represents the current stack trace.
    4. Let realm be the current Realm Record.
    5. Let strategy be realm.[[ErrorCaptureStackTraceStrategy]].
    6. If strategy is data-property, then
      1. Perform ? DefinePropertyOrThrow(O, "stack", PropertyDescriptor { [[Value]]: string, [[Writable]]: true, [[Enumerable]]: false, [[Configurable]]: true }).
    7. Else,
      1. Assert: strategy is accessor.
      2. If ? IsExtensible(O) is false, throw a TypeError exception.
      3. Define a new [[StackTrace]] internal slot on O.
      4. Perform ? DefinePropertyOrThrow(O, "stack", PropertyDescriptor { [[Get]]: %ErrorCaptureStackTraceGetter%, [[Set]]: %ErrorCaptureStackTraceSetter%, [[Enumerable]]: false, [[Configurable]]: true }).
    8. Return undefined.

    20.5.2.2 %ErrorCaptureStackTraceGetter% ( )

    The value of %ErrorCaptureStackTraceGetter% is a built-in function that takes no arguments. It performs the following steps when called:

    1. Let O be the this value.
    2. If O has a [[StackTrace]] internal slot, then
      1. Return O.[[StackTrace]].
    3. Else,
      1. Return undefined.

    20.5.2.3 %ErrorCaptureStackTraceSetter% ( value )

    The value of %ErrorCaptureStackTraceSetter% is a built-in function that takes an argument value. It performs the following steps when called:

    1. Perform ? DefinePropertyOrThrow(this value, "stack", PropertyDescriptor { [[Value]]: value, [[Writable]]: true, [[Enumerable]]: true, [[Configurable]]: true }).
    2. Return undefined.
